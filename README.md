## Задание 2

Проведите сканирование Metasploitable в режимах SYN, FIN, Xmas, UDP.

Запишите сеансы сканирования в Wireshark.

Ответьте на следующие вопросы:

Чем отличаются эти режимы сканирования с точки зрения сетевого трафика?
Как отвечает сервер?
Приведите ответ в свободной форме.

Ответ

Чем отличаются эти режимы сканирования с точки зрения сетевого трафика?

SYN-сканирование: Nmap отправляет TCP-пакет с флагом SYN, открытый порт отвечает SYN/ACK, закрытый - RST/ACK.

FIN-сканирование: Отправляется TCP-пакет с флагом FIN, открытые порты игнорируют такие пакеты, закрытые отвечают RST.

Xmas-сканирование: Аналогично FIN, но отправляется пакет с флагами FIN, PSH и URG.

UDP-сканирование: Отправляются пустые UDP-пакеты. Закрытые порты отвечают ICMP "Port Unreachable", открытые либо не отвечают, либо отправляют данные.

Как отвечает сервер?

На SYN: открытые порты - SYN-ACK, закрытые - RST На FIN/Xmas: открытые - нет ответа, закрытые - RST На UDP: закрытые - ICMP ошибка, открытые - молчание или UDP-ответ
